🎬 Streama

Streama est une application permettant de gérer et accéder à vos contenus multimédias facilement.

🚀 Installation
Téléchargez le fichier .zip depuis la page Releases
Extrayez le fichier .zip
Ouvrez le dossier Streama
Laissez le fichier .exe dans le dossier
Lancez l’application (si nécessaire, exécutez en tant qu’administrateur)
📌 Utilisation
Une fois l’application ouverte, vous pouvez créer un compte
Choisissez une offre :
Offre de base (gratuite)
Ou une offre payante selon vos besoins
Connectez-vous et commencez à utiliser l’application
🛠️ Support

En cas de problème :

Vérifiez que vous avez bien extrait le fichier .zip
Lancez le programme en administrateur si besoin
Ouvrez une issue sur GitHub si le problème persiste
🎯 Démarrage

Une fois l’installation terminée et le compte créé :
👉 c’est parti, profitez de Streama !
